## ‚úÖ **Clarified Requirements**

### **Organization System**

- **Main Collection:** Every user has one primary collection
- **Sub-collections:** Optional organizational folders (e.g., "Family Guy", "Star Wars")
- **Tags:** Flexible labels for detailed categorization within any collection level
- **User Flexibility:** Some users will use only tags, others will use sub-collections + tags

### **Privacy & Value Tracking**

- **Purchase Price Tracking:** Optional field users can choose to track/display
- **No Complex Valuation:** Focus on what users paid, not current market value
- **Field-Level Privacy:** Most tracking fields can be set as visible/hidden to others
- **Collection Privacy:** Collections can be private/public with granular sharing

### **Following System**

- **User Following:** Follow users to see their public collections
- **Collection Following:** Follow specific public collections for updates
- **Dual Notifications:** Users get updates from both followed users and collections

### **Platform Focus**

- **Core First:** Bobblehead management and sharing takes priority
- **Future Content:** Stories/posts can be added later as community grows
- **Mobile Simplicity:** Quick-add with photo, name, basic tags
- **Desktop Power:** Full management, analytics, detailed editing
